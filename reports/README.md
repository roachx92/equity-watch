# Deep-dive reports

Full-diligence deep-dive reports live here as **markdown in git** — this repo is the
source of truth. Each report is a **point-in-time snapshot** in a dated folder:

```
reports/<YYYY-MM-DD>/<TICKER>.md      e.g. reports/2026-07-14/IBIDY.md
```

A new deep-dive is a new dated file, so history is kept as discrete snapshots rather than
diffs of one mutable file. Reports are generated by running the framework on a ticker; the
binding standard is [`framework/deep-dive-template.md`](../framework/deep-dive-template.md)
(Sections B/C/D/G/H) under [`framework/standing-rules.md`](../framework/standing-rules.md).

`_template.md` is the empty skeleton to copy from; it is not published to the site.

_Not financial advice — informational research tooling only._
